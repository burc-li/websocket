# websocket

## 简介
websocket API 目录：用一个小DEMO 简单介绍 websocket 的API

websocket 目录：对 websocket 进行封装，支持断网重连、自动心跳等功能

## websocket API

**安装依赖**
```
npm install
```

**启动node服务**
```
nodemon websocketAPI/node.js
```
**浏览器打开html文件**

浏览器打开 websocketAPI/websocket.html，F12 查看 NetWork面板

tip：可以使用 vscode中的 Live Server 插件
